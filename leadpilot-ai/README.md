# 🚀 LeadPilot AI

AI-powered lead response & follow-up system for real estate agents.

## 💡 Problem
Most real estate agents lose deals because:
- Slow response to leads
- No proper follow-up system
- Leads get ignored or forgotten

## ⚡ Solution
LeadPilot AI automates the entire lead handling process:

- 📥 Capture leads from WhatsApp
- ⚡ Instant auto-response
- 🧠 Extract key info (budget, location)
- 📊 Store & track leads
- 🔁 Automated follow-ups

## 🛠️ Tech Stack
- Node.js (Backend)
- Express.js
- Supabase (Database)
- WhatsApp Cloud API (Meta)
- OpenAI (planned)

## 🔄 Workflow
1. User sends message on WhatsApp
2. Backend captures lead
3. Auto-reply is sent instantly
4. Data is stored in Supabase
5. Follow-ups triggered automatically
