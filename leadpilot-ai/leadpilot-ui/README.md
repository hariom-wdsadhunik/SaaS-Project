# LeadPilot UI

Frontend dashboard for LeadPilot AI - built with Stitch design tool.

## Files

- **code.html** - Main dashboard HTML with embedded JavaScript
  - Real-time lead display
  - Status update dropdowns
  - Auto-refresh every 10 seconds
  - Material Design styling

## Features

- View all leads from Supabase
- Update lead status (new, contacted, follow-up, closed)
- Color-coded status badges
- Responsive design
- Mobile-friendly

## API Connection

Connects to backend at `http://localhost:80` (or ngrok URL when shared)

## Access

Local: `http://localhost:80/dashboard`
Public (via ngrok): `https://your-ngrok-url.ngrok-free.dev/dashboard`
